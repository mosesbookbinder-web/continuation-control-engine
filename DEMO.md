# Demo

The purpose of the demo is to show a minimal reproducible continuation-control evaluation.

## Demo Flow

1. provide a sample artifact
2. compute its hash
3. evaluate the artifact
4. emit a deterministic decision
5. verify the result

## Expected Output

Artifact SHA256: <hash>  
Decision: PASS  
Kernel decision: PASS  
Witness chain: VALID

## Demo Goal

A user should be able to clone the repository, run a demonstration command, and inspect a reproducible decision artifact.
